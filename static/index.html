api API 文档

认证方式

所有接口需通过以下两种方式之一认证：

  • 浏览器：自动使用登录 Cookie，无需额外操作。
  • 命令行 / curl：在请求头加 Authorization: Bearer <API_KEY>。API Key 默认与访问密码相同，也可在 config.py 中单独设置 API_KEY 字段。

返回格式

          
所有接口均返回 JSON。成功时包含 "ok": true 或对应数据字段；失败时包含 "error": "..." 及对应 HTTP 状态码（400 / 401 / 404 / 409 等）。
